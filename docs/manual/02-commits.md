## Commit Message Regex

`(feat||fix|docs|style|refactor|test|chore)\(.*\)\:.*\`

## Prefix

- feat
- fix
- docs
- style
- refactor
- test
- chore

and add `:`

## Message

Lowercase letters

## Example

feat: organization of items
